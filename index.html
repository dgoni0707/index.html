Git commands

Git commands

1. fetch

Команда git fetch used to download commits and files from a remote repository to your repository

2. merge

Команда git merge used to merge two branches

3. rebase

Команда git rebase used to merge two branches

The difference between them is what REBASE compresses all changes, and then inserts them into the main branch, removing unnecessary branches

4. pull

Команда git pull - this is an automated version of the git fetch command
